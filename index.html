O mně

Jsem junior web developer s vášní pro čistý a sémanticky správný kód. Když jsem začínala svou kariéru, fascinoval mě způsob, jak HTML, CSS a JavaScript spolupracují na vytváření interaktivních zážitků. Nyní to je moje profese a nadále mě to inspiruje.
